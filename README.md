# The Fastify Workshop

![ci](https://github.com/nearform/the-fastify-workshop/workflows/ci/badge.svg)

## Requirements

- Node LTS
- yarn - you can install it with `npm i -g yarn`
- docker
- docker-compose

## Setup

- `yarn`
- `yarn db:up`
- `yarn db:migrate`

### Run automated tests

- `yarn test`

## Running the modules

- `cd src/step-{n}-{name}`
- check each module's README file to see which scripts are available

## Presenting

- `yarn start`
